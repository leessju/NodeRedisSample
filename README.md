# RedUsers
Simple user management app using Node.js and Redis

### Version
1.0.0

## Usage

Make sure you have Redis installed and running

Install Dependencies

```sh
$ npm install
```

Run Server

```sh
$ npm start
```

Visit http://localhost:3000
